# Student Database — SQLite with Functions

## 📌 Description
A student database system using Python and SQLite, built with structured functions.

## ⚙️ Features
- Add student (name, age, grade)
- View all students
- Update student by ID
- Delete student by ID
- Data stored in SQLite database

## 🛠️ Tech Used
- Python
- SQLite3

## ▶️ Run
```
python main.py
```
